# DevToolBox

Free, privacy-first developer utilities. All tools run 100% client-side.

## Tools (MVP)

- **JSON Formatter & Validator** — Beautify, minify, validate JSON
- **Regex Tester** — Real-time regex matching with highlighting
- **Base64 Encoder/Decoder** — UTF-8 aware encode/decode
- **JWT Decoder** — Decode tokens, inspect claims, check expiry

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel (free tier)

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub, then connect to Vercel. Zero config needed.

## Environment Variables

See `.env.example` for optional analytics/ad configuration.
